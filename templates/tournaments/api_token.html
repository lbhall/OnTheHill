Create a tournament via API

POST /api/tournaments/ with a JSON body. Teams are registered as individual entrants in the order given. 

curl -X POST http{% if request.is_secure %}s{% endif %}://{{ request.get_host }}/api/tournaments/ \
  -H "Authorization: Token YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Friday Night 9-Ball",
    "game_type": "9ball",
    "format": "double_elim",
    "entry_fee": 20,
    "teams": ["Alice", "Bob", "Carol", "Dave"]
  }'

Optional fields: date (YYYY-MM-DD), added_money, venue_id, notes. Each team may also be an object: {"name": "Alice", "email": "...", "phone": "..."}.
